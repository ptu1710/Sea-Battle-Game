# Sea-Battle-Game

Đồ án môn học Lập trình mạng căn bản - NT106

Mô hình nhiều Client - 1 Server, có database lưu dữ liệu tài khoản người dùng

Giao tiếp qua mạng LAN, có thể phát triển thành mạng Internet.
